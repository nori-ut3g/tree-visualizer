# Pruner

## 特徴

Prunerはその名の通り、木の剪定、つまり木構造（二分木）の挿入や削除を可視化することができます。

木構造の表示は階層順にならべた配列を入力するだけです。

[demo](https://u3g.github.io/Pruner/)
でチェックしてください。

## Requirement

- anime.js

## Usage

1. npm経由でインストール
   
2. ブラウザ

## Installation

```
$ npm install pruner

```
```
$ git clone https://github.com/u3g/Pruner

```
