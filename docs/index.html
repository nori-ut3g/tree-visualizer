背景

Prunerはその名の通り、木の剪定、つまり木構造を切ったり挿入したりを可視化することができます。

コンピューターサイエンスを学習していて、どうしても木構造のイメージが沸かない。。。

試しに前順走査などを実行しようとしたが、ちゃんと前順走査になっているかパット見わからない。。。

可視化することで、学習の手助けをします。

特徴

可視化する上で、どうしても必要な操作があります。

木構造（二分木）を表現するにはどうしても、階層順で配列にして入力する必要があります。

とはいえ、例を準備したのでまずは実行してみてください。
