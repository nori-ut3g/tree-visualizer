1. 二分木の表示 階層表示された文字列から二分木を可視化します。 同じ値が存在する場合は、ノードごとにIDを設定する必要があります。 4.Optionを確認してください。 例1 : [0,-10,5,null,-3,null,9] 例2 : [50,17,76,9,23,54,null,null,14,19,null,null,72]
2. Animation 階層表示された文字列を連続して入力すると、差分を自動で計算しAnimationを生成します。 Animationの詳細の設定は3.setting AnimationはAnime.jsを使用しています。
3. Setting
boxColor
textColor
arrowColor
animationInterval
boxSize
boxXMargin
boxYMargin
4. Option 同じ値が存在する場合は、IDを加える。 色を変えることもできる。
